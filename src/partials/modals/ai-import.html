透過 AI 建立流程

1. 點擊按鈕複製這段提示詞 (Prompt)，並貼上給您的 AI (如 ChatGPT、Claude)。 

請幫我設計一套伸展運動菜單。
請直接輸出 JSON 格式，不要包含 Markdown 標籤或其他文字解說。
請依照以下格式：
{
  "name": "菜單名稱",
  "description": "菜單整體描述 (如果有需要準備的道具，請寫在這裡)",
  "steps": [
    {
      "name": "動作名稱",
      "duration": 秒數(單次動作要持續的時間, 整數),
      "repeat": 組數(動作要重複的次數, 整數),
      "bilateral": 是否為單側動作需要換邊(布林值, 若為 true，請統一以右側來撰寫動作名稱與解說，系統會自動產生換左側的流程),
      "description": "動作解說(可選，請用引導並簡短的語氣，請在適當的地方使用 \n 換行)"
    }
  ]
}

2. 將 AI 產生的 JSON 貼在下方，並點擊「匯入並編輯」。
